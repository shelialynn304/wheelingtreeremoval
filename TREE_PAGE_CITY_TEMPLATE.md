# TREE PAGE TEMPLATE

This file defines the exact structure for all tree service pages that have a location file. (City or state)

Codex should follow this template when:

- Creating new city/location pages
- Rewriting existing location pages

Goal:
Every page should look consistent, rank locally, and convert visitors into calls or form leads.

---

## PAGE STRUCTURE (REQUIRED)

### 1) HEAD SECTION

Each page MUST include:

- Unique <title>
- Unique meta description
- Canonical URL
- Viewport meta
- Link to style.css

Example:

<title>Tree Removal in [City, State] | Free Estimates</title>

<meta name="description" content="Professional tree removal in [City]. Fast, affordable, and reliable service. Call now for a free estimate.">

<link rel="canonical" href="https://yourdomain.com/[page-url].html">

<link rel="stylesheet" href="style.css">

---

### 2) NAVIGATION

Use the SAME nav across all pages.

Required links:
- Home
- About
- Tree Removal
- Stump Grinding
- Emergency
- Free Quote (anchor)
- Call Now button (tel link)

DO NOT redesign nav per page.

---

### 3) HERO SECTION (ABOVE THE FOLD)

Must include:
- H1 with service + city
- Short supporting text
- Strong CTA button (call)
- Optional secondary CTA (quote)

Example:

H1: Tree Removal in [City, State]

Subtext:
Fast, affordable, and professional tree service. Free estimates.

CTA buttons:
- Call Now
- Get Free Quote

---

### 4) TRUST / INTRO SECTION

Short paragraph:
- Who you are
- What you do
- Why you’re reliable

Keep it real. No fake claims.

---

### 5) SERVICES SECTION

Break into clear chunks:

- Tree removal
- Emergency removal
- Stump grinding
- Storm cleanup

Each gets:
- short explanation
- benefit-driven language

---

### 6) WHY CHOOSE US

Bullet points:

- Fast response
- Local service
- Fair pricing
- Safe work practices

Keep it simple and believable.

---

### 7) COST / VALUE SECTION

Explain pricing in general terms:
- depends on size
- depends on location
- depends on risk

Optional:
- link to cost calculator page

DO NOT give fake exact prices.

---

### 8) FAQ SECTION (IMPORTANT FOR SEO)

Include 4–6 questions like:

- How much does tree removal cost in [City]?
- Do you offer emergency service?
- How quickly can you come out?
- Do you remove large/dangerous trees?
- Do you clean up after?

Answers should be:
- clear
- local
- not generic fluff

---

### 9) FINAL CTA SECTION

Strong closing push:

- Call now
- Free estimate
- Fast response

Make this obvious and hard to ignore.

---

### 10) CONTACT / FORM SECTION

If present:
- name
- phone
- message

Short and clean.

---

## SEO RULES

- Only ONE H1 per page
- Use H2 for sections
- Naturally include:
  - "tree removal in [city]"
  - "tree service [city]"
- Do NOT keyword spam
- Write for humans first

---

## LOCALIZATION RULES

Every page MUST:
- mention the actual city
- NOT be a copy-paste clone
- have slightly different wording

Bad:
Same page with city swapped

Good:
Feels like it was written for that area

---

## DESIGN RULES

- Use existing classes from style.css
- Do NOT inline huge CSS blocks
- Keep layout consistent with other pages
- Mobile-first always

---

## CTA RULES (VERY IMPORTANT)

Every page MUST have:
- Call button near top
- Call button near bottom
- Quote CTA somewhere mid-page

---

## INTERNAL LINKS

Include links to:
- Homepage
- Other service pages

---

## WHAT TO AVOID

- Fake reviews
- Fake credentials
- Keyword stuffing
- Huge paragraphs
- Walls of text
- Random design changes
- Breaking existing layout

---

## DEFAULT PAGE TYPE

Unless specified otherwise, assume:
- This is a local SEO landing page
- The goal is CALLS, not just traffic

---

## SUCCESS CRITERIA

A good page:
- Looks like the rest of the site
- Loads fast
- Reads easy on mobile
- Clearly pushes user to call
- Feels local and real
