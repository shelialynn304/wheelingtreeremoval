# TREE PAGE CHECKLIST

Use this checklist BEFORE considering any page complete.

If a page fails multiple checks, it should not be published.

---

## BASIC PAGE SETUP

- [ ] Page uses template-tree-page.html
- [ ] Links to style.css
- [ ] No massive inline CSS blocks
- [ ] No massive inline JS blocks
- [ ] Page loads without layout breaking

---

## SEO CORE

- [ ] Unique <title> (includes service + city)
- [ ] Unique meta description (not copied)
- [ ] Canonical URL is correct
- [ ] Only ONE H1 on page
- [ ] H1 includes service + city
- [ ] Proper H2/H3 structure (not random)

---

## LOCALIZATION (CRITICAL)

- [ ] City name appears naturally throughout page
- [ ] Content is NOT a lazy copy/paste with city swapped
- [ ] Intro paragraph feels specific to the area
- [ ] FAQ mentions city where appropriate

If this section fails → page is basically useless for SEO

---

## CONVERSION (THIS IS WHAT MAKES MONEY)

- [ ] Call button near top (above the fold)
- [ ] Call button near bottom
- [ ] Quote CTA exists mid-page
- [ ] CTA text is clear ("Call Now", "Get Free Quote")
- [ ] Page makes it obvious what to do next

If user has to “figure out” what to do → you lose the lead

---

## CONTENT QUALITY

- [ ] No giant walls of text
- [ ] Sections are broken up clearly
- [ ] Writing sounds human, not robotic
- [ ] No keyword stuffing
- [ ] No fake claims (years, awards, certifications)

---

## SERVICES SECTION

- [ ] Tree removal included
- [ ] Emergency service included
- [ ] Stump grinding included
- [ ] Storm cleanup included
- [ ] Each has a short, clear explanation

---

## COST SECTION

- [ ] Explains pricing depends on:
  - size
  - location
  - difficulty
- [ ] Does NOT give fake exact pricing
- [ ] Pushes toward quote request

---

## FAQ SECTION (SEO BOOST)

- [ ] At least 4 questions
- [ ] Answers are clear and useful
- [ ] Includes local phrasing
- [ ] Not generic filler content

---

## FORM / LEAD CAPTURE

- [ ] Form works (correct FormSubmit or endpoint)
- [ ] Name field present
- [ ] Phone field present
- [ ] Message field present
- [ ] Submit button visible and clear

---

## TRACKING SAFETY

- [ ] Existing analytics scripts not removed
- [ ] Call buttons keep class: call-now-btn
- [ ] Quote buttons keep class: quote-btn
- [ ] Form has ID: quote-form

Breaking this = you lose data without realizing it

---

## MOBILE CHECK (VERY IMPORTANT)

- [ ] Buttons are easy to tap
- [ ] Text is readable (not tiny or huge)
- [ ] No elements overlapping
- [ ] No weird empty gaps
- [ ] Hero section looks clean on phone

If mobile sucks → you lose most leads

---

## LINKS

- [ ] Internal links to:
  - homepage
  - at least 1 service page
- [ ] Nav links work
- [ ] No broken links

---

## FINAL QUALITY CHECK

Ask:

- [ ] Would a real person trust this page?
- [ ] Would I call this number if I needed help?
- [ ] Does this look like a real local business?

If answer is “eh…” → fix it

---

## HARD FAIL CONDITIONS (DO NOT PUBLISH)

If ANY of these are true:

- [ ] Duplicate page with only city swapped
- [ ] No CTA near top
- [ ] No form or contact option
- [ ] Broken layout
- [ ] Missing title or meta description
- [ ] Looks spammy or fake

→ DO NOT SHIP
